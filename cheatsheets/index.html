Активная учёба

Не просто читай — проверяй себя

Режим теста превращает карточки в тренажёр: задание показывается первым, а решение открывается только когда ты сам готов.

  • Ответ по запросу. Сначала думаешь сам, потом открываешь решение.
  • Самооценка «Знал / Не знал». Отмечаешь результат и видишь итог в конце.
  • Гибкий охват. Тренируйся по всем темам, по избранному или по одному предмету.
